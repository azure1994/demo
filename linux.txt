Folder creation commands
------------------------

mkdir -> create folder

ex: mkdir demo

ls -> it is used to check list of folders or files

ls -lart -> it is used to check list of folders or files with hidden

cd - to open the folder

pwd -> to check location or path

cd .. -> come one step back from current location

cd ../.. -> come two steps back from current location

File creation commands
----------------------

touch -> to create file

ex: touch devops

vi -> to edit the file or to open file

i -> INSERT mode (to write anything)

esc :wq! -> Save and quit from file

esc :q! -> without Save and quit from file

cat -> to view the file
